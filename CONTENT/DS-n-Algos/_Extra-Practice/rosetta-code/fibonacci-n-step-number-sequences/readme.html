Fibonacci n-step number sequences

FCC link

These number series are an expansion of the ordinary Fibonacci sequence where:

  1. For n = 2 we have the Fibonacci sequence; with initial values $\[1, 1\]$ and F_k2 = F_k − 12 + F_k − 22
  2. For n = 3 we have the tribonacci sequence; with initial values $\[1, 1, 2\]$ and F_k3 = F_k − 13 + F_k − 23 + F_k − 33
  3. For n = 4 we have the tetranacci sequence; with initial values $\[1, 1, 2, 4\]$ and F_k4 = F_k − 14 + F_k − 24 + F_k − 34 + F_k − 44…
  4. For general n > 2 we have the Fibonacci n-step sequence - F_kn; with initial values of the first n values of the (n − 1)’th Fibonacci n-step sequence F_kn − 1; and k’th value of this n’th sequence being $F\_k^n = \\sum\_{i=1}^{(n)} {F\_{k-i}^{(n)}}$

For small values of n, Greek numeric prefixes are sometimes used to individually name each series.

Allied sequences can be generated where the initial values are changed: The Lucas series sums the two preceding values like the fibonacci series for n = 2 but uses $\[2, 1\]$ as its initial values.
