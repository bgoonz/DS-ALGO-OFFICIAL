# Factorial

[FCC link](https://www.freecodecamp.org/learn/coding-interview-prep/rosetta-code/factorial)

Write a function to return the factorial of a number.

Factorial of a number is given by:

n! = n \* (n-1) \* (n-2) \* ..... \* 1

For example:

- `3! = 3 * 2 * 1 = 6`
- `4! = 4 * 3 * 2 * 1 = 24`

**Note:** `0! = 1`
